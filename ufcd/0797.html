UFCD 0797 - Sistemas Operativos - Tipologias

Nesta UFCD falámos em sistemas operativos, cuja função é gestão de todo o hardware e software inseridos no equipamento. O tipo de sistema operativo varia consoante o equipamento e interface a que nos referimos. Por exemplo pode ter uma interface por Linha de Comandos, em que para ser utilizados é necessário conhecimento técnico para a sua utilização, normalmente utilizado em máquinas industriais. Pode também conter uma interface Gráfica, que é uma equipamento mais userfriendly, mais fácil de utilizar e intuitivo, como um computador. Por fim pode ter também uma interface Móvel/Voz geralmente usado em equipamentos móveis como os smartphones e tablets.

Falando em topologia dos sistemas operativos temos os Monotarefa, que executam apenas uma tarefa decada vez, ou seja, para passarmos à tarefa seguinte temos de fechar a tarefa anterior. Neste sentido temos os Monoutilizadores, como computadores pessoais, em que apenas usa um utilizador de cada vez. Depois temos os sistemas Multitarefa, como os computadores, onde podemos fazer várias tarefas em simultâneo e neste sentido os sistemas Multiutilzador, que são sistemas em que vários utilizadores podem trabalhar em simultâneo cada um com o seu login de acesso. Temos também os sistemas Centralizados, em que funcionam a partir de um servidor, os sistemas Embarcados, que são feitos para equipamentos com uma utilização específica, por exemplo um tensiómetro, tem um sistema embarcado cuja função é apenas avaliar a tensão arterial de uma pessoa. Temos também os sitemas em Tempo Real que se divide em 3 tipos, o Duro, em que tem de dar uma resposta imediata, caso não funcione haverá danos graves, o Intermédio, em que a resposta pode ter um delay que os danos não serão significativos e o Suave, em que o tempo de reação não é importante, o importante é que reaja. Por fim temos os sistemas Móveis, como Android e IOS usados em tablets e smartwathes

Antes de instalarmos um sistema operativo ou qualquer tipo de software, convém verificarmos primeiro se o nosso equipamento tem os requisitos mínimos para que possa funcionar corretamente. Para instalar o Windows 11, primeiro que tudo devemos fazer um backup de toda a informação para que não arrisquemos a perder dados importantes Depois podemos fazer de 2 formas, ou através do nosso próprio disco interno, em que fazemos download do instalador do windows, no próprio site da microsoft, e quando o executamos podemos Atualizar o sistema operativo que já temos e desta forma normalmente não perdemos a informação, porque apenas os ficheiros do sistema operativo serão alterados, ou então podemos instalar de forma Personalizada/Avançada, que nos permite fazer uma instalação limpa, todo o disco será apagado e toda a informação será perdida. Desta forma também eliminamos possiveis virus e malwares que possam estar entranhados no nosso computador.

De igual forma também podemos instalar a partir de periféricos externos, como uma pen, DVD ou disco externo para isso temos de criar um ficheiro de arranque no periférico pretendido e depois na BIOS/UEFI do nosso computador temos de alterar esse periférico como prioridade 1 no Boot, para que o computador quando ligar entre pelo boot desse periférico. Após o arranque pelo periférico, é tudo feito da mesma forma que anteriormente referi, ou atualizamos ou escolhemos uma instalação personalizada/avançada.

Na pós instalação do windows temos de verificar se existem atualizações do sistema, caso existam, devemos intalar, tal como devemos instalar as drivers e configurar os periféricos necessários e personalizar o ambiente de trabalho, barra de tarefas, de ferramentas ,etc conforme as necessidades do utilizador, para que não estejamos a entupir o sistema com configurações e softwares que o utilizador não vai utilizar. Após tudo feito, temos de ir fazendo sempre backups regulares para nunca corrermos o risco de perder dados importantes, convém irmos sempre atualizando o sistema operativo, que por vezes fazem atualizações de segurança muito importantes para manter o nosso computador seguro. Fora isso, devemos ter cuidado com os softwares que instalamos, devemos recorrer a softwares de fontes fidedignas e ter sempre cuidado com o links que abrimos ou sites que visitamos, para não apanharmos malware ou phishing.

Esta matéria será essencial tanto a nível pessoal como profissional, porque é importante perceber como se instala um S.O. e como se prepara o computador para se poder utilizar, consoante as nossas necessidades. A nível de futuro na área de programação é fulcral perceber como tudo funciona para que possamos fazer o código da forma mais acertada e limpa possível.

Eu já trabalhei numa loja de informática e já executei estes procedimentos muitas vezes, quer para os meus próprios computadores, quer para clientes e sempre ajustei o Pós Instalação do S.O. consoante as necessidades dos clientes.
